# Description
An HTTP service that functions as a bridge, enabling clients to interact with Web3. 
Currently, it only supports StarkNet.


# Usage 
## Installation
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
